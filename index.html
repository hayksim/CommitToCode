Troubleshooting: so as I pasted my project from SublimeText to codepen I encountered some issues. Problem: First of I had an image that had to be linked to the codepan page. After googling for a while I found a forum topic discussing linking it to GitHub account, that seemed to be the fastest.

Solution: Upload your project files into GitHub account (create a new repository), go the image of interest, open it, right click on the image and choose Inspect. Then go to Sources, right click on the image and choose Copy image as Data URI. After that paste the link into your codepen src of the image (random numbers and letters starting with data:image/ ).

Problem: offset in bootstrap did not work.

Solution: when working in SublimeText I used bootstrap 3.6. After some research I found out that bootstrap 4 is using the offset differently. Instead of col-*-offset-number it should be offset-*-number. Or .img-responsive is now .img-fluid. Will watch some videos and do some reading on what has changed in Bootstrap 4.
